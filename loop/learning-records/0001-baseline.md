# Baseline: 에이전트 도구 사용자, 루프 미구현

사용자는 Claude Code 등 에이전트/자동화 도구를 **써본** 경험은 있으나, 루프를 직접
코드로 구현해본 적은 없다. 도구·자동화 개념은 익숙하므로 "에이전트가 뭔가"부터가 아니라
"루프를 어떻게 직접 짜는가"에서 시작하는 것이 ZPD에 맞다.

## Implications
- 기초 개념 강의는 짧게, 곧바로 직접 구현(코드 실행)으로 넘어갈 것 (학습 선호: 직접 만들기).
- 미션이 "내 에이전트/자동화 제작"이므로 모든 레슨은 실행 가능한 코드 → 실제 업무 적용으로 수렴.
- Lesson 01에서 최소 루프(gather/act/verify/repeat)와 verify의 중심성을 다룸. 아직 *demonstrated*
  아님(coverage). 사용자가 verify 개념을 스스로 적용하면 그때 GLOSSARY로 승격하고 LR 추가.
